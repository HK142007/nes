# Nes
Nes Emulator for meteor application.


Install and Run
=======

    $ git clone https://github.com/yasaricli/nes.git
    $ cd nes
    $ meteor
    

Import a Games 
=============

    $ mongorestore -h 127.0.0.1:3001 --db meteor .backups/02112015

# Nes
Nes Emulator for meteor application.

Follow progress on Libreboard here [Nes board][Nes].

Install and Run
=======

    $ git clone https://github.com/yasaricli/nes.git
    $ cd nes
    $ meteor

Uses
=======
A JavaScript NES emulator from [JSNES][Jsnes], thank you [bfirsh][Bfirsh]
